# Portfolio Images

This folder contains images for the portfolio case studies.

## Required Images for King Case Study

Please export the following images from the King case study PDF and save them here with these exact filenames:

1. **king-common-challenges.png** - Slide showing common developer challenges
2. **king-ux-vision.png** - UX Vision slide with empowering platform experience
3. **king-ux-pillars.png** - Five UX Pillars (Effortless, Self-Explanatory, etc.)
4. **king-future-scenario.png** - Future scenario with Tiffy character
5. **king-ux-canvas-workshop.png** - UX Canvas workshop illustration
6. **king-ux-playbook.png** - UX Design Methods Playbook overview

## Image Requirements

- Format: PNG (preferred) or JPG
- Resolution: High quality (at least 1200px wide recommended)
- File naming: Use exact names listed above (case-sensitive)

## How to Export from PDF

1. Open the PDF in Preview or Adobe Acrobat
2. Navigate to the slide you want to export
3. File → Export → Select PNG format
4. Save with the corresponding filename from the list above
5. Place all images in this folder

Once all images are saved, refresh the King case study page to see them displayed.
